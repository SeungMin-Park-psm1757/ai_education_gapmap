# 데이터 출처와 실제 데이터 연결 원칙

## 핵심 원칙

이 MVP는 임시자료를 사용하지 않습니다. API 키나 실제 CSV 파일이 없으면 화면은 더미값을 보여주지 않고 “실데이터 준비 필요” 상태를 표시합니다.

## 필수 교육 공공데이터

| 데이터 | 역할 |
|---|---|
| NEIS 학교기본정보 | 학교명, 주소, 전화번호, 홈페이지, 교육지원청 기준 |
| 학교알리미 공시정보 | 학생 수, 교원 수, 시설, 교육활동, 학교 현황 |
| 교육통계서비스 | 지역·학교급·학생·교원 통계 보완 |
| 교육청 AI·디지털 교육 사업 공개자료 | AI 교육 운영 신호 |
| 학교 시설·정보화 관련 공시자료 | 디지털 인프라 지표 |

## 보조 데이터

| 데이터 | 역할 |
|---|---|
| 지역 인구·소득·도시/농산어촌 구분 | 지역 여건 보정 |
| 공공 SW·AI 교육센터 위치 | 지역 지원 접근성 |
| 교육청 연수·교원연수 자료 | 교원 역량 지원 신호 |
| OECD·UNESCO 등 국제 자료 | 지표 설계 참고 |
| 외국교육기관 및 외국인학교 종합안내 | 외국인학교 교원 여건·시설·운영 신호 보조 |
| 국무조정실·교육부 보도자료 | 학력인정 평생교육시설 학생·학급·교직원 현황 보조 |

## 현재 자동 수집 항목

- NEIS `schoolInfo`
- 공공데이터포털 `전국초중등학교위치표준데이터`
- 학교알리미 Open API `62`, `09`, `22`, `17`, `18`, `56`, `59`

학교알리미 공시가 별도 기관명으로 제공되지 않는 부설기관은 동일 주소 본교 공시자료를 보조 근거로 반영합니다. API 공시 대상 밖의 외국인학교·학력인정 평생교육시설은 `data/source/public-supplements.json`에 공식 공개자료 근거를 분리해 기록합니다.

## 표현 원칙

- “학교 평가”가 아니라 “지원 우선순위 진단”으로 표현합니다.
- “AI가 학교를 평가한다”가 아니라 “공공데이터 기반으로 지원 필요 신호를 정리한다”고 표현합니다.
- 실제 수업 운영 수준은 공개자료만으로 확정하지 않습니다.
