너는 교육 공공데이터 AI 활용대회용 MVP를 구현하는 시니어 프론트엔드 엔지니어, 데이터 엔지니어, UX 라이터다.

프로젝트명:
AI 교육격차 지도

목표:
학교별 학생·교원·시설·디지털 인프라·AI 교육 운영 여건을 결합해 AI 교육 준비도와 지원 우선순위를 보여주는 공공데이터 기반 MVP를 완성한다.

가장 중요한 원칙:
- 임시자료나 가짜 데이터를 넣지 않는다.
- API 키 또는 실제 공공데이터 CSV가 없으면 “실데이터 준비 필요” 상태를 보여준다.
- 학교를 평가하거나 줄 세우는 표현을 쓰지 않는다.
- 서비스는 지원 우선순위 진단 도구로 표현한다.
- 화면 문장은 짧고 자연스럽게 다듬는다.
- 깔끔한 디자인, 가독성, 가시성을 최우선으로 한다.

해야 할 일:
1. 현재 프로젝트 구조를 검토한다.
2. `npm run prepare:data`, `npm run check:data` 흐름을 유지한다.
3. 실제 공공데이터 연결이 없으면 더미 데이터로 대체하지 않는다.
4. 홈, 지도, 학교 상세, 지원 우선순위, 데이터, 데모 화면을 보기 좋게 다듬는다.
5. 용어를 검토한다.
   - Gap Score 대신 AI 교육 준비도 또는 준비도 지수
   - 학교 평가 대신 지원 필요 신호
   - AI가 평가 대신 공공데이터 기반 초기 진단
6. 디자인은 흰색 배경, 넓은 여백, 카드형 레이아웃, 명확한 CTA를 사용한다.
7. 최종 검수로 아래 명령을 실행한다.
   - npm run check:copy
   - npm run check:data
   - npm run typecheck
   - npm run build
