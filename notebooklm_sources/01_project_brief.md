# 01. Project Brief

## 서비스명

AI 교육격차 지도

## 한 줄 정의

AI 교육격차 지도는 공개자료로 학교별 AI 교육 지원 필요 신호를 산출하고, 교육청이 예산·연수·프로그램 지원을 어디에 먼저 검토할지 돕는 의사결정 MVP이다.

## 해결하려는 문제

교육청은 한정된 예산과 인력으로 여러 학교를 지원해야 한다.

기존 학교정보 조회만으로는 다음 질문에 빠르게 답하기 어렵다.

- 어느 학교에 디지털 학습공간·기기 점검이 먼저 필요한가
- 어느 학교에 AI 활용 수업 연수와 수업설계 컨설팅이 필요한가
- 어느 학교에 방과후 AI·SW 프로그램이나 지역 SW교육센터 연계가 필요한가
- 어떤 학교는 점수 해석보다 데이터 보완과 현장 확인이 먼저 필요한가

이 MVP는 흩어진 공공데이터를 결합해 1차 지원 필요 신호를 보여준다.

## 주요 사용자

| 사용자 | 필요 |
| --- | --- |
| 교육청 정책 담당자 | 예산·연수·프로그램 지원 우선순위 검토 |
| 교육지원청 장학·행정 담당자 | 학교별 보강 영역과 현장 확인 대상 파악 |
| 학교 관리자 | 공개자료상 나타난 지원 신호와 추가자료 필요 항목 확인 |
| 공모전 심사위원 | 공공데이터 활용, AI 활용, 정책 효과를 짧은 시간 안에 검토 |

## 핵심 가치

1. 공개자료 기반으로 1차 지원 대상을 빠르게 좁힌다.
2. 점수에서 끝나지 않고 다음 행정 조치로 연결한다.
3. 데이터 신뢰도를 총점과 분리해 과잉 해석을 줄인다.
4. AI 디지털교과서(AIDT)·학습관리시스템(LMS) 자료는 총점에 섞지 않고 2단계 확장진단으로 둔다.
5. 제출용 익명화 모드로 실제 학교 식별정보를 노출하지 않는다.

## 지원 소요 진단 원칙

이 서비스는 학교를 평가하거나 서열화하는 도구가 아니다.

`지원 소요 지수`는 점수가 높을수록 좋은 점수가 아니라, 공개자료상 AI 교육 지원 필요 신호가 크다는 뜻이다.

따라서 결과 해석은 다음 관점으로 해야 한다.

- 우수·미흡 판단이 아니라 지원 우선 검토
- 개별 학교 책임 판단이 아니라 행정 지원 필요 신호
- 자동 결론이 아니라 현장 확인과 추가자료 요청의 출발점

## 현재 MVP 범위

| 항목 | 내용 |
| --- | --- |
| 분석 단위 | 서울 A권역 학교 98개 |
| 제외 기준 | 특수목적 성격으로 일반 비교가 어려운 학교 유형 2개 제외 |
| 1단계 지수 | 공개자료 기반 지원 소요 지수 |
| 2단계 확장 | AI 디지털교과서(AIDT)·학습관리시스템(LMS) 자료 연계 시 운영 안정성 별도 진단 |
| 제출 모드 | 학교명, 주소, 연락처, 홈페이지, 실제 좌표 익명화 |
