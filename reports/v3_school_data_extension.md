# V3 School Data Extension

작성일: 2026-05-28

## 1. 목적

v3는 기존 v2의 화면 구조를 유지하면서 데이터 산출값만 바꾼 확장 버전이다.

- v2: 공개자료 기반 지원 소요 지수
- v3: 공개자료 + 학교 제공 추가자료 예시 기반 지원 소요 지수

## 2. 추가한 학교 데이터 예시

학교가 교육청에 다음 자료를 제공한다고 가정했다.

| 항목 | 의미 |
| --- | --- |
| AIDT 접속 안정성 | 접속 성공률, 장애 빈도, 가동 안정성 |
| LMS 사용 지속성 | 주간·월간 사용빈도, 수업 적용 지속성 |
| 교원 연수 이수 | AI 활용 수업 연수와 컨설팅 이력 |
| 기기 접근성 | 학생 기기 접근 가능성과 수업 활용 가능성 |
| AI·SW 프로그램 커버리지 | 방과후·동아리·지역 연계 프로그램 운영 정도 |
| 추가자료 완전성 | 학교 제출자료의 결측 정도 |

## 3. 산출 방식

v3는 v2의 공개자료 신호를 버리지 않는다.
학교 추가자료 예시를 결합해 다음 영역을 다시 계산한다.

- 교원·운영 여건: 교원 연수 이수 자료 반영
- 디지털·학습공간 기반: AIDT 접속 안정성, 기기 접근성 반영
- AI·SW 학습기회: LMS 사용 지속성, 프로그램 커버리지 반영
- 지역 지원 접근성: 외부 프로그램 접근 자료 반영

## 4. 현재 v3 산출 결과

| 구분 | 학교 수 |
| --- | ---: |
| 우선 지원 | 15 |
| 보완 검토 | 72 |
| 일반 모니터링 | 11 |
| 현장 확인 우선 | 0 |
| 합계 | 98 |

## 5. 해석

v3는 실제 기관 연계 전의 시연용 산출이다.
목적은 “학교 추가자료가 들어오면 더 정밀한 판단이 가능하다”는 구조를 보여주는 것이다.

따라서 v3 값은 최종 정책 판단이 아니라 확장 가능성 검증용이다.

## 6. 실행 방법

```bash

NEXT_PUBLIC_DEMO_ANONYMIZE=true NEXT_PUBLIC_DATA_VERSION=v3 npm run start -- -p 3003
```
