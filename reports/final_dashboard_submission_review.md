# Final Dashboard Submission Review

작성일: 2026-05-27

## 1. 반영 요약

- 제출용 익명화 모드 `NEXT_PUBLIC_DEMO_ANONYMIZE=true`를 추가했다.
- 상단 메뉴에서 별도 설명 화면을 제거하고 `/demo`는 홈으로 redirect한다.
- 홈 화면을 `무엇인가 / 공공데이터 / AI 활용 / 교육청 결정` 4개 카드로 재구성했다.
- 지도, 우선순위, 데이터, 상세 화면 상단에 점수 방향 안내를 추가했다.
- `/priorities`와 상세 리포트의 행정 조치 카드를 `왜 필요한가 / 무엇을 할 것인가 / 필요한 추가자료` 구조로 강화했다.
- `/data` 화면에 활용 데이터 표, 전처리·정규화, 산식, 직접지표·대리지표, AI 활용 위치, 한계와 2단계 확장진단을 추가했다.

| 화면 | 제출 관점 |

| `/` | 문제의식, 데이터, AI 활용, 정책 결정을 첫 화면에서 확인 |

| `/priorities` | 학교별 보강 영역과 다음 행정 조치를 확인 |

| `/data` | 공공데이터 출처, 산식, AI 활용, 한계를 확인 |

## 5. 남은 한계

- AIDT·LMS 자료는 현재 총점에 반영하지 않고 `확장진단 대기`로 둔다.
- 공개자료 기반 1단계 지수는 교실 안 AI 활용의 질을 직접 확정하지 않는다.
- 신뢰도 C 사례는 현재 데이터에 없어 조건부 로직으로 검증한다.
- 실제 배포 시 제출 환경의 `NEXT_PUBLIC_DEMO_ANONYMIZE=true` 설정을 다시 확인해야 한다.
