# AI 교육격차 지도 MVP v2

교육 공공데이터 AI 활용대회용 MVP입니다.

- GitHub: <https://github.com/SeungMin-Park-psm1757/ai_education_gapmap>
- 시연 시작 화면: `/map`
- 현재 버전: v2, 별도 설명 화면 제거 및 격차지도 중심 구성

## 한 줄 설명

AI 교육격차 지도는 공개자료로 학교별 AI 교육 지원 필요 신호를 찾고, 교육청이 예산·연수·프로그램 지원을 어디에 먼저 투입할지 돕는 의사결정 MVP입니다.

## 왜 필요한가

교육청은 한정된 예산과 인력으로 여러 학교를 지원해야 합니다. 기존 학교정보 조회만으로는 어느 학교에 디지털 기반, 교원 연수, AI·SW 프로그램, 지역 연계를 먼저 검토해야 하는지 한눈에 보기 어렵습니다.

이 MVP는 흩어진 공공데이터를 결합해 학교별 `지원 소요 지수`를 만들고, 지도와 상세 리포트에서 다음 행정 조치까지 연결합니다.

## 현재 분석 범위

- 지역: 서울특별시 노원구
- 대상: 초·중·고 및 공시 기반 비교가 가능한 학교 98개
- 제외: 학력인정 평생교육시설 2개교
- 제외 이유: 특수목적 성격의 학교로 일반 초·중·고 지원 소요 비교에 함께 넣으면 해석이 왜곡될 수 있음

## 제출용 익명화

`.env.example`에는 다음 설정이 포함되어 있습니다.

```env
NEXT_PUBLIC_DEMO_ANONYMIZE=true
```

`true`일 때 UI는 실제 학교명, 주소, 전화번호, 홈페이지, 구체 지역명을 표시하지 않습니다. 학교명은 `A초`, `B중`, `C고` 형식으로 바뀌고, 지역은 `서울 A권역`으로 표시됩니다. 데이터 출처 문서와 산식 설명은 공공데이터 활용 사실을 유지합니다.

## 핵심 지표

`지원 소요 지수`는 점수가 높을수록 우수하다는 뜻이 아닙니다. 공개자료상 AI 교육 지원이 먼저 필요한 신호가 크다는 뜻입니다.

총점은 100점 만점이며 다음 5개 영역을 반영합니다.

| 영역 | 가중치 | 의미 |
| --- | ---: | --- |
| 교육 수요 | 25 | 학생 규모, 학급 규모 등 지원 수요 신호 |
| 교원·운영 여건 | 20 | 교원 배치와 운영 여건 보강 신호 |
| 디지털·학습공간 기반 | 20 | 기기, 무선망, 학습공간 관련 보강 신호 |
| AI·SW 학습기회 | 20 | 방과후, 동아리, AI·SW 프로그램 접근 신호 |
| 지역 지원 접근성 | 15 | 학교 밖 프로그램과 지역 자원 접근 신호 |

데이터 신뢰도는 총점에 넣지 않고 A/B/C 배지로 분리합니다. 신뢰도 C는 점수가 높아도 `현장 확인 우선`으로 별도 표시합니다.

## 주요 화면

| 경로 | 역할 |
| --- | --- |
| `/` | 서비스 목적, 분석 학교 수, 지원 구간 요약 |
| `/map` | 우선 지원 TOP 3, 지도 기반 지원 소요 분포, 범례 |
| `/priorities` | 학교별 지원 소요 지수, 대표 보강 영역, 추천 행정 조치 |
| `/schools/[id]` | 학교 상세 리포트, 점수 근거, 다음 행정 조치, 2단계 확장진단 예시 |
| `/data` | 데이터 출처, 산식, 직접지표와 대리지표, 추가자료 필요 항목 |

v2에서는 별도 설명 화면을 노출하지 않습니다. 격차지도(`/map`)만 봐도 점수와 지원 우선 대상이 바로 보이도록 동선을 단순화했습니다.

## 데이터 구성

1단계 지수는 공개자료 기반입니다.

- NEIS 학교 기본정보
- 학교알리미 공시자료
- 교육통계 및 교육청 공개자료
- 학교 위치·시설·프로그램 공개자료

2단계 확장진단은 아직 총점에 반영하지 않습니다. AIDT·LMS 자료가 제공되면 접속 안정성, 장애 시간, 사용 지속성 등을 별도 진단 슬롯으로 확장할 수 있습니다.

## 실행 방법

```bash
npm install
cp .env.example .env.local
npm run prepare:data
npm run dev
```

로컬 실행 후 다음 주소에서 확인합니다.

```text
http://localhost:3000/map
```

## 검증 명령

```bash
npm run check:copy
npm run typecheck
npm run build
```

## 대회 발표 포인트

- 개별 학교의 우열을 가르는 서비스가 아니라 지원 필요 신호를 찾는 도구입니다.
- 공개자료로 1차 지원 대상을 좁히고, 추가자료가 제공되면 2단계 확장진단으로 정밀화합니다.
- 지도에서 우선 지원 TOP 3를 바로 확인하고, 상세 리포트에서 예산·연수·프로그램·현장 확인 조치로 연결합니다.
